투자 대시보드

주요 시장 지수, NPS 자산배분, SEC 13F 이력, 시장 심리 지표를 한 화면에서 확인합니다.

이 대시보드의 데이터 출처

이 화면은 서로 다른 공식 소스를 합쳐서 보여줍니다. 각 섹션은 수집 기준과 범위가 다릅니다.

  • 시장 지수: KIS API (KOSPI/KOSDAQ), yfinance (NASDAQ/S&P500/Gold), CoinGecko (Bitcoin)
  • 시장 심리: data.go.kr KOFIA 통계 API (신용잔고/투자자예탁금)
  • NPS 자산배분: 국민연금 기금운용본부 월간 공시 (공식 페이지)
  • 국내 보유종목(KOSPI/KOSDAQ): OpenDART 전자공시에서 국민연금공단 관련 대량보유/주요주주 보고를 집계 (OpenDART)
  • 국내 평가금액/가중치: KIS 시세 API 키가 있으면 평가금액 기준, 없으면 공시 지분율 정규화 기준으로 계산
  • 미국 보유종목: SEC 13F (NPS CIK 기준)로 미국 상장 주식 일부만 포함 (SEC EDGAR)

참고: 새로고침을 누를 때마다 최신 데이터를 다시 조회하며, 공시 지연/공개 범위에 따라 실제 운용잔고와 시차가 있을 수 있습니다.
